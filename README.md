# Welcome to the Mood Store

## Description
* Simple demo store created using HTML/CSS/JavaScript
* Features include
    * Modal popup for each item
    * Add items to cart
    * Increase/Reduce Quantity
    * Remove items from cart
* [Link to repo](https://github.com/HelixHEX/mood-shop)
